# Caliber

A modern platform for evidence-based student assessment and mastery.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Components**: [Lucide React](https://lucide.dev), [Retro Grid](https://magicui.design/docs/components/retro-grid)

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when pushing to the `main` branch.
The site is served at [www.caliber.cs.ucsb.edu](http://www.caliber.cs.ucsb.edu).
